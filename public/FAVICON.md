# Neo ERP Favicon

## Archivos incluidos

- **favicon.svg** - Favicon principal en formato SVG (escalable, moderno)
- **favicon.ico** - Favicon de respaldo para navegadores antiguos (opcional)
- **site.webmanifest** - Manifiesto PWA para instalación en dispositivos móviles

## Diseño

El favicon de Neo ERP combina dos elementos:

1. **Letra "N"** estilizada en blanco - Representa "Neo"
2. **Icono de caja/paquete** en la esquina - Representa el aspecto de gestión empresarial (ERP)

### Colores
- **Fondo**: Gradiente azul (#3498db → #2980b9) - Profesional y tecnológico
- **Letra N**: Blanco (#ffffff) - Contraste y legibilidad
- **Caja**: Tonos de gris y blanco - Profundidad y dimensión

## Cómo actualizar el favicon

Si necesitas crear un favicon personalizado:

1. **Editar favicon.svg**: Modifica el archivo SVG directamente
2. **Generar favicon.ico**: Usa herramientas online como:
   - https://realfavicongenerator.net/
   - https://favicon.io/
3. **Actualizar colores**: Modifica los valores hex en el SVG y site.webmanifest

## Notas técnicas

- El SVG es compatible con todos los navegadores modernos
- Se incluye meta theme-color para Android/Chrome
- El webmanifest permite instalar como PWA
- Los navegadores priorizan SVG sobre ICO cuando está disponible
